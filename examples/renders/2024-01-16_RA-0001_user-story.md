### RA-0001 — Exclude out-of-stock products from the recommendation carousel without shrinking the catalog page
**As a** Recommendation service (owner: Product Owner), **I want** products flagged out-of-stock removed from the recommendation carousel (product = SKU × store), **so that** the carousel hides unavailable items while the catalog page still shows the full assortment.

**Acceptance criteria (Given/When/Then)**
- Given a carousel render, When a SKU is out-of-stock, Then it is omitted from the carousel.
- _(negative)_ Given a new SKU with unknown stock, When the carousel renders, Then it is **NOT** excluded.
- _(negative)_ Given an exclusion is applied, When it runs, Then the catalog page product count is **unchanged**.
- **Fallback:** if stock status is unknown (new SKU) — do not exclude (keep).
- **Must never:** change the number of products shown on the category/catalog page.
